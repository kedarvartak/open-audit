# Decentralized Fund Release & Verification System

## Monorepo Structure

- **apps/web**: React + Tailwind CSS Frontend.
- **apps/backend**: NestJS Backend.
- **apps/engine**: Python AI Service (Gemini 3.0 Pro).
- **packages/contracts**: Solidity Smart Contracts.
- **packages/config**: Shared configurations.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```
